# auto_play_Tower_of_Saviors
# 自動刷神魔素材
這是我一年半前寫來給自己的隊伍(奇牙帶小傑)刷素材用的      
基本上這東西是利用 PC版的TeamViwer + 手機板的 "TeamViewer QuickSupport" + sikulix(v1.1.2)去使用      
目前神魔已退坑，所以不會再更新此專案，但發給各位參考看看。
